Debating Society, IIEST Shibpur - Official Website
This repository contains the source code for the official static website of the Debating Society (DebSoc), IIEST Shibpur. The website serves as a digital hub for information regarding the society's history, upcoming events, and debating resources for the student community.

Project Overview
The website is designed to be a lightweight, fast-loading platform that showcases the society's activities. As a static site, it focuses on clean presentation and ease of maintenance.

Tech Stack
The project is built using fundamental web technologies to ensure compatibility and performance:

HTML5: Used for structuring the website's content.

CSS3: Used for styling, layout (Flexbox/Grid), and responsive design to ensure the site works on all device sizes.

JavaScript (Vanilla): Used for DOM manipulation, interactive elements (such as navigation toggles or sliders), and minor animations.

Key Features
Responsive Design: Fully optimized for desktops, tablets, and smartphones.

Event Gallery: A section dedicated to past tournaments and debating sessions.

Member Directory: Information about the current executive body and active members.

Resources Section: Links and documents to help budding debaters learn different formats (British Parliamentary, Asian Parliamentary, etc.).

Local Development
To run this project locally on your machine, follow these steps:

Clone the Repository

Bash
git clone https://github.com/debsoc-iiests/official-website.git
Navigate to the Directory

Bash
cd official-website
Open the Project
Simply open the index.html file in any modern web browser, or use a local development server like Live Server in VS Code for real-time updates.
